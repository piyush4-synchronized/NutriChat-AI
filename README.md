# AI Clinical Dietitian 🥗

This project is a fully conversational, AI-driven nutrition tracker and virtual dietitian. Built using **Python, Streamlit, and Groq (Llama 3.3)**, it goes beyond basic calorie counting. The app features smart onboarding to calculate baseline energy needs, parses natural language for real-time meal logging via the **USDA API**, and uses **Retrieval-Augmented Generation (RAG)** to browse live internet guidelines to recommend highly personalized, medically-safe meals for conditions like Type 2 Diabetes and PCOS.
